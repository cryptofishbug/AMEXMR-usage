# 파트너 로고

- **로컬에 파일이 없으면** [Clearbit Logo](https://clearbit.com/logo) API로 브랜드 로고를 자동 로드합니다. (원본에 가까운 깔끔한 로고)
- **직접 넣고 싶으면** 아래 파일명으로 PNG를 넣으면 됩니다. (`{partner id}.png`)

## 파일명 (partner id)

- **항공**: `cathay.png`, `flyingblue.png`, `finnair.png`, `airasia.png`, `delta.png`, `singapore.png`, `vietnam.png`, `koreanair.png`, `eva.png`, `turkish.png`, `etihad.png`, `hainan.png`, `aeroplan.png`, `qatar.png`, `united.png`, `jal.png`
- **호텔**: `hilton.png`, `marriott.png`, `ihg.png`, `all.png`, `wyndham.png`

배경 제거(remove.bg 등)한 이미지는 깨질 수 있어, 원본 로고를 쓰는 것을 권장합니다.
